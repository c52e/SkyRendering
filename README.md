# SkyRendering

## Requirements

* Windows 10
* Visual Studio 2019

## Build

```
git clone git@github.com:c52e/SkyRendering.git
cd SkyRendering
git submodule update --init
```
Open SkyRendering.sln.

Set SkyRendering as startup project.

Build and run.
